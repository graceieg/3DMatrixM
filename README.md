# 3D Matrix Manipulation

Interactive 3D matrix transformations built with React and Three.js.

## Quick Start

```bash
git clone https://github.com/yourusername/3DMatrixM.git
cd 3DMatrixM
cd "3D Matrix Manipulation Frontend"
npm install
npm run dev
```

Open `http://localhost:3000`

## Features

- 3D transformations (translate, rotate, scale)
- Real-time visualization
- Interactive 3D viewport
- Matrix manipulation panel
